| 6   | [fault_slip.def](fault_slip.def) | 断层滑动符号   | 汪园园  |
| 7   | [focal_mec.def](focal_mec.def) | 震源机制符号   | 汪园园  |
| 8   | [BS.def](BS.def) | 钻孔槽符号   | 汪园园  |
